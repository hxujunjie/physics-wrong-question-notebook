教师端 v1.6.2（多服务商智能识别 · 导入 JSON 路径优化 · 确认后生成错题集）

正式交付包（exe）：release\v1.6.0_教师端_多服务商智能识别_未压缩交付\教师端_v1.6.0\
开发/源码启动：本目录「一键启动教师端.bat」

1. 双击“一键启动教师端.bat”（或桌面上的“物理错题整理-教师端”快捷方式）。
   首次运行会创建 .venv、安装依赖，并自动在桌面生成一键启动图标；以后只做快速自检。
2. 若桌面没有图标，可双击“创建桌面快捷方式.bat”。打包版 exe 首次打开也会自动创建同名桌面快捷方式。
3. 服务只监听 127.0.0.1。已在运行时再次启动不会创建第二个服务，只会重新打开浏览器页面。
   若刚更新了代码，启动器会检测版本；旧进程不会被静默复用。
4. 首次打开时在页面粘贴 API Key 并点击“保存”。软件会按密钥形态自动填入服务商、默认模型与接口地址；也可用下拉框手动选择：
   - 智谱：id.secret（中间有点）→ glm-4.6v-flash（免费视觉）
   - 通义：sk- → qwen2.5-vl-3b-instruct（低价）
   - OpenAI/GPT：sk-proj- → gpt-4o-mini
   - Google Gemini：AIza… → gemini-3.5-flash（新用户；2.5 常不可用）
   - xAI Grok：xai- → grok-2-vision-1212
   - 豆包：ark- 开头（需填接入点 ID）
   模型请在「模型」下拉框中选择（含免费/低价标记）；选「其他/自定义」可手填型号。
   密钥只使用 Windows 当前用户加密保存。
5. 若自动识别不准，可改服务商/模型下拉框后再次保存（不必重贴 Key）。
6. 在页面选择干净练习册 PDF、学生照片总目录（每个学生一个子文件夹）和可选输出根目录。软件会扫描学生、显示调用预算，再开始识别。
6b. 若用「导入 JSON」代替在线识别：同样选择干净 PDF + 学生照片总目录，再选择识别结果 JSON，点「导入识别结果」（无需 API Key）。
   JSON 中 photo_file 可只写文件名；软件按「照片总目录/学生姓名/文件名」自动绑定。导入后仍须复核，再点「生成错题集」。
7. 识别完成后：
   - 高置信结果会自动填好；
   - 不确定 / 低置信题需要老师确认；
   - 可切换「整页查看」（可跳过）；
   - 老师修改只记录，不会立刻生成 PDF；
   - 最后点击「生成错题集」才导出个人错题 PDF。
8. 使用页面“退出服务”正常停止。若页面无法打开，可用“维护工具\停止教师端.bat”；
   “维护工具\重新安装依赖.bat”仅用于修复依赖。

安全边界：识别时会将压缩后的 PDF 页面和学生照片通过所选 AI 服务商 API 发送；原始文件不会上传为长期云端文件。每次启动生成新的会话令牌，令牌仅注入当前首页；所有修改、路径选择和停止接口均验证 X-Review-Token。照片读取仅允许 manifest 声明的路径，静态资源与照片路径穿越会被拒绝。
