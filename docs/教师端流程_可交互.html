老师最短路径

  1. 启动教师端，粘贴 API Key 并测试连接
  2. 选择干净 PDF + 学生照片目录，开始识别
  3. 只处理待确认题；页码错了改页眉；横图可旋转
  4. 只有裁不出 / 裁错 / 跨页跨栏时才手动框选
  5. 返回首页，点击「生成错题集」
